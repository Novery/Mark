# 内部类
+ 静态内部类可以有静态成员，而非静态内部类则不能有静态成员。 
+ 静态内部类的非静态成员可以访问外部类的静态变量，而不可访问外部类的非静态变量

# 值类型间的自动转型
+ 可以这样来定义一个double型的
```
  double d = 0.1;
```
+ 然而，这样不可以通过
```
  float f = 0.1;
```
+ 来定义一个float型的，这是因为默认的小数是double型的
+ 通常，表达式最终的数据类型是最大的决定的 一个int型加一个short型 最后结果是int型 如果要把值赋给较小的类型，就需要使用造型
```sql
byte 1字节
char 2字节
short 2字节
int 4字节
float 四字节
long 八字节
double 八字节
```
